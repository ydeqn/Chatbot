# Chatbot
 Chatbot Prototyp für ein WAGP Projekt der TH Köln. Die Dateien sind nur für das Verzeichnis des Codes gelistet. Funktionalität mit dem Code erfordert noch entsprechende Umgebungen um zu funktionieren. Bei Bedarf kann die rag-desktop.exe ausgeführt werden zum testen

# Nützliche Informationen
Dies ist ein Prototyp. Es sind noch viele Verbesserungsmöglichkeiten offen. Zu dem ist die Programm Optimierung noch nicht erfolgt.
Das Projekt basiert auf dem Tauri Framework und wurde mit Typescript und React fürs Front-End, erstellt. Durch das Tauri Framework kann es schwierig sein relevante Dateien zu finden. Dafür sind die Pfade hier gelistet:

-> Chatbot/Front-End/Chatbot/rag-desktop/src/App.css 
------
-> Chatbot/Front-End/Chatbot/rag-desktop/src/App.tsx
------

Back-End wurde außerhalb des Tauri Frameworks entwickelt in der AWS (Amazon Web Services) Umgebung mit Lambda, S3 und API-Gateway
